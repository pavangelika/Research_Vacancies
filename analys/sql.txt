Для каждой роли создай вкладку мои отклики.
Мои отклики содержат таблицу с вакансиями с теми же полями и функционалом, что во вкладке поиск по навыкам.
Только в поле отклик вместо слова отклик и ссылки будет чек-бокс. С текущим значением параметра send_resume из get_vacancies для данного id
Справа от поля отклик, дата отклика, собес. В поле дата отклика вывести resume_at.
В поле собес вывести ссылку подробнее. При нажатии на ссылку будет выводится таблица с полями:
requirement, responsibility, description, published_at, archived, archived_at
и формой ввода:
поле в БД - имя в интерфейсе
hr_name - ФИО HR
interview_date - Дата и время собеседования
interview_stages - Этапы собеседования
company_type - Тип компании
result - Результат собеседования
feedback - Обратная связь
offer_salary - Сумма оффера
pros - Плюсы
cons - Минусы
введенные данные сохранять в get_vacancies в соответсвующие поля по ключу ID.
При открытии в следующий раз показать введенные ранее данные, при их повторном
изменении запрашивать у пользователя разрешения перезаписи данных в БД.

Поля требования и обязанности убрать из таблицы.
