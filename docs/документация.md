	Структура фильтров:
	Мои фильтры: избранное и сохранение выбранных фильров
	Фильтры роли: Роли, Исключить роли, Период, Опыт, Статус

	Мои фильтры - Избранное. Выбор из списка доступных значений.
	Мои фильтры - Сохранение.
	
	Фильтры роли - Роль. Доступны функции: поиск роли, выбрать все, мультивыбор, очистить все. 
	Выбранные роли перемещаются вверх списка, подписано количество выбранных ролей. Выбранные роли отображаются в результатах выдачи.
	Выбрано больше одно ролей. Выбрано: количество.
	
	Фильтры роли - Исключить роли. Доступны функции: поиск роли, выбрать все, мультивыбор, очистить все. 
	исключенные роли перемещаются вверх списка, подписано количество исключенных ролей. Исключенные роли не отображаютмся в результатах выдачи.
	Исключено больше одно ролей. Исключено: количество.
	
	Роли и Исключить роли не может быть выбрана одна и та же роль в обеих списках.
	При выборе роли в одном списке она блокируется для выбора в другом. 
	
	Фильтры роли - Период. Варианты: Сегодня, за 3 дня, за 7 дней, за 14 дней, 2026-04, 2026-03, ... за ...количество месяцев. 
    Доступны функции: мультивыбор, очистить все. Выбрать все - отстутствует. 
	По умолчанию выбирать за последний месяц. 
	
	Фильтры роли - Опыт. Варианты: Нет опыта, От 1 года до 3 лет, От 3 до 6 лет, Более 6 лет.
	Доступны функции: выбрать все, мультивыбор, очистить все. По умолчанию выбрать все. 
	
	Фильтры роли - Статус. Варианты: Открытая, Архивная.
	Доступны функции: выбрать все, очистить все. 
	По умолчанию выбрать все. 
	
	Фильтры зарплаты - Валюта. Варианты: RUR, USD, EUR, Другая, Не заполнена. 
	Доступны функции: выбрать все, мультивыбор, очистить все. 
	По умолчанию выбрать все. Вывести как Не выбрана

	Фильтры зарплаты - Страна. Варианты: Россия, не Россия.
	Доступны функции: выбрать все, очистить все. 
	По умолчанию выбрать все. Вывести как Не выбрана
	
	Фильтры зарплаты - чипсеты. Мин, Средн, Медиана, Мода, Макс. 
	По умолчанию среднее.
	
	Фильтры откликов - Собес назначен. Варианты: Назначен, Не назначен.
	Доступны функции: выбрать все, очистить все. 
	По умолчанию выбрать все. Вывести как Не выбрано
	
	Фильтры откликов - Результат указан. Варианты: Указан, Не указан.
	Доступны функции: выбрать все, очистить все. 
	По умолчанию выбрать все. Вывести как Не выбрано
	
	Фильтры откликов - Оффер. Варианты: Получен, не Получен.
	Доступны функции: выбрать все, очистить все. 
	По умолчанию выбрать все. Вывести как Не выбрано
	
	Фильтры вакансии - Работодатель. Список работодателей.
	Доступны функции: поиск, выбрать все, мультивыбор, очистить все. 
	По умолчанию выбрать все. Вывести как Не выбрано. 

	Фильтры вакансии - ИТ-аккредитация. Варианты: Есть, Нет.
	Доступны функции: выбрать все, очистить все. 
	По умолчанию выбрать все. Вывести как Не выбрано
	
	Фильтры вакансии - Сопроводительное письмо. Варианты: Требуется, Не требуется.
	Доступны функции: выбрать все, очистить все.

Календарь требования:
  отображается текущий месяц, текущий день недели, текущая дата
  месяц и год можно сменить
  дни с собесами выделены в календаре цветом
  собесы отображаются как количество
  есть панель фокус на дне, ближайшие 7 дней, не внесен результат собеса
  Если время собеса истекло и пользователь не обновил данные по собесу, то собес отображается в разделе не внесен результат собеса.
  Если сегодня есть собес, то собес отображается в Фокус дня, в ближайшие 7 дней не отображается.
  Если в фокус дня показаны собесы, то не дублировать их в ближайшие 7 дней
  Выделены зеленым цветом как все собесы.
  Клик по плашке собеса открывает модульное окно для внесения информации по собесу
  Кнопки закрыть - не сохраняет изменения даже при внесенных данных, кнопка сохранить - сохраняет данные.
  Календарь не фильтруется по фильтрам роли, фильтрам навыков, фильтрам вакансии
  Календарь фильтруется по фильтрам зарплаты, фильтрам откликов
